О программе


  Данная Программа (графическая оболочка и данное Руководство) принадлежит Крезуб П. Цитирование, изменение и перепечатка, распространение только с упоминанием авторства Крезуб П.
  Программа предоставляется по принципу «как есть». Используйте её на Ваш страх и риск. Автор Программы снимает с себя всякую ответственность за какой-либо вред, причиненный данной Программой Вам, вашему оборудованию (компьютеру) и/или документам (файлам).
Искренне Ваш © Крезуб П., 2016 г.

  При написании Программы ни одно животное не пострадало!

  Требования к компьютеру – Pentium III, 512Mb RAM, 2Gb свободного места на жестком диске, на котором расположен каталог временных файлов (как правило это системный диск С:\). При конвертации PDF и TIFF преобразуется в несжатый растр.

  Программа "Конвертер" является унифицированной оболочкой, включающей в себя модули и библиотеки
с бесплатным и условно-бесплатным доступом, т.е. ТОЛЬКО для личного использования.
Запрещено использование программного продукта и/или его модулей в коммерческих целях.

В программе использованы следующие модули:
- GhostScript (gsdll64.dll, gsdll64.lib)
- NConverter (nconvert64.exe)
- PdfTk (pdftk.exe, libiconv2.dll)
- Ildar Shaimordanov (doc.bat, doc2fb.xsl)

  Всплывающее меню выбора выходного формата открывается и закрывается
последовательным нажатием на любую кнопку формата.
  Кодовая страница изменяется в верхнем меню и служит для правитьного понимания компьютером посылаемых ему команд в обработчик cmd.exe. Если функции не работают, то проверьте номер установленной кодовой страницы. По умолчанию установлена страница 1251, используемая в Windows, в случае ошибок поменяйте на 866, используемой IBM, DOS и Windows.

  Функция Drag-and-Drop (пер. "тащи и бросай"):
- Можно сжимать JPEG JPG, PDF, TIF или TIFF файлы просто перетащив их в главное окно программы и отпустив над блоком Сжать.
- НЕ РЕКОМЕНДУЕТСЯ перетаскивать одновременно несколько файлов одного или различных форматов любое из полей окна программы, т.к. при этом возможно появление ошибки, которая может повредить документы.
- Качество сжатых изображений по умолчанию будет равным 50, если необходимы файлы с еще меньшим объемом, то необходимо воспользоваться кнопочными функциями программы для сжатия данных форматов.
- Также поддерживается конвертирование поддерживаемых программой "Конвертер" файлов перетаскиванием их в главное окно программы и отпустив над блоком Конвертировать из. Далее автоматически определяется тип файла и появляется меню выбора выходного формата файла и, в зависимости от выбранного формата, для некоторых форматов окно дополнительных параметров.
- При перенесении более одного файла формата jpg в блок конвертирование, файлы автоматически объединяются в один файл pdf.
- При перенесении более одного файла формата pdf в блок сжатие, файлы автоматически объединяются(склеиваются) в один файл pdf.
