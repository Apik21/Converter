Сжатие



Формат TIFF

 - Необходимо выбрать сжимаемый файл *.tiff
- В появившемся окне выбирается качество изображения выходного файла,
чем меньше качество, тем легче файл. Качество варьируется от 1 до 99.
По умолчанию качество выходного файла 85.
- Тут же можно поставить галочку для удаления исходного файла после сжатия и создания нового.

Формат PDF

Разрезать:

 - Собирает файл с именем <имя файла>New.pdf
- В поле страницы задается список страниц включаемых в новый файл.
- Страницы записываются через пробел (БЕЗ ЗАПЯТОЙ).
- Четность возможно указать как even (четный, 2-20even) или odd (нечетный, 1-13odd)
- Поворот указывается как north, south, east, west, left, right или down.
- Последнюю страницу документа можно указать при помощи ключевого слова end.
- Диапазон указывается или прямо, как в примере, или реверсивно (например, end-1)
- Можно комбинировать ключи: 2-6evendown - выберет четные страницы и перевернет их на 180.

Склеить:

 - Позволяет выбрать несколько файлов *.pdf и объединить их в один выходной файл *.pdf.

Сжать:

 - Необходимо выбрать сжимаемый файл *.pdf
- В появившемся окне выбирается качество изображения выходного файла,
чем меньше качество, тем легче файл. Качество варьируется от 1 до 99.
По умолчанию качество выходного файла 85.
- Тут же можно поставить галочку для удаления исходного файла после сжатия и создания нового.

Восстановить:

 - Производится попытка восстановления поврежденного PDF файла

Изменить метаданные:

 - Необходимо выбрать файл *.pdf метаданные которого будут редактироваться.

Защита:

 - Выводится диалоговое окно с запросом Установки или Снятия защиты с PDF файла
- Если выбрано "ДА", то будет произведена защита PDF файла от копирования, печати, экспорта страниц и прочих изменений.
Защита ставится по придуманному Вами паролю защиты. ВАЖНО не забывать и не терять этот пароль, восстановить его невозможно.
- Если выбрано "НЕТ", то будет произведено снятие защиты по ранее введенному паролю защиты.
