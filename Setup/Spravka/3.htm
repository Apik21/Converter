Формат PDF

 - Принимает файлы изображений с расширением *.pdf и конвертирует в форматы *.jpg, *.tiff (Цвертной), *.tiff (Черно-белый),*.djvu, *.txt.
- Конвертирование в txt возможно только в том случае, если PDF файл содержит текстовый слой. Иначе команда возвращает пустой файл *.txt.

Формат TIFF

 - Принимает файлы изображений с расширением *.tiff и конвертирует в формат *.jpg, *.pdf, *.ps, *.png, *.ico, *.emf, и *.bmp.
- Во всплювающем меню можно выбрать качество выходногофайла при конвертации, от 1 до 99
- Так же можно выбрать номер определенной страницы (кадра) файла TIFF, если оставить 0, то конвертируются все страницы многостраничного файла.
- Имеется возможность выбора удаления исходных файлов после конвертирования.

Формат DOC


- Принимает файлы документов - DOC, DOCX, HTML, XML, RTF, MHT, TXT.
- Конвертирует исходные файлы в - FB2, PDF, XPS, HTML, XML, RTF, MHT, TXT.
- Во избежании ошибок, единовременно могут быть конвертированы файлы одного формата.
- Для корретного отображения кирилицы конвертируемый файл должен быть сохранен в кодировке UTF-8.
