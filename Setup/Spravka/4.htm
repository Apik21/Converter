ДАТА


- Данный раздел позволяет изменять свойства времени любого файла;
- Поддерживается выбор и изменение нескольких файлов;
- Вначале необходимо ввести согласно шаблона новое время и дату;
     - Шаблон формата: YYYYMMDDHH24MISS,
где HH24 - это формат времени, т.е. должно получиться так: 20160711100805
YYYY - 2016 - год, ММ - 07 - месяц, DD - 11 - число, 10 - HH24 - часы, MI - 08 - минуты, SS - 05 - секунды.
     - Годы до 1601 не поддерживаются;
- Далее выбираем какой пункт свойства (время изменения/создания/последнего доступа) файла нужно изменить;

ШИФРОВАНИЕ


- Данный раздел позволяет зашифровать/дешифровать любой файл;
- Шифрование производится по алгоритму Base64;
- Зашифрованный файл замещает исходный файл;
- Дешифрованный файл замещает исходный файл;
- Для шифрования папки, можно сделать её архивом zip или любым другим и шифровать уже файл архива.
- Кнопка Шиф./Дешиф. текст запускает программу в которой можно вручную набрать шифруемый / дешифруемый текст для преобразования "налету";
- В всплывающем меню кнопки "Шифрование" есть поля для ввода количества циклов повтора(интераций) операции шифрования или дешифрования. Таким образом если файл был зашифрован за 10 интераций, то чтобы его расшифровать потребуется произвести так же 10 интераций дешифрования.;
- Если заранее не известно за сколько интераций был зашифрован файл, то рекомендуется последовательно проводить дешифрование с количеством циклов повтора равным 1, до тех пор пока файл не будет расшифрован;
- После каждого цикла размер файла увеличивается.
