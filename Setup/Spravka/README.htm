Программа представляет собой скрипт, который в одном окне позволяет использовать ниже перечисленные программы для работы с файлами изображений.
The program is a script that allows you to use the following programs in one window to work with image files.

  • GhostScript (gsdll64.dll, gsdll64.lib)
  • NConverter (cvert64.exe, cvert.exe)
  • PdfTk (pdftk.exe, libiconv2.dll)
  • Ildar Shaimordanov (doc.bat, doc2fb.xsl)
